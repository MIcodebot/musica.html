# 🎵 musica.html

Página de **Podcast/Lofi** desenvolvida com **HTML5 e CSS3**, com reprodução de áudio, estilo visual personalizado e imagem de fundo temática.

---

## 📋 Sobre o Projeto

Este repositório contém o código-fonte de uma página de podcast com tema **Lofi**, criada para praticar o uso de HTML e CSS. Inclui estilização com CSS inline, elemento de áudio nativo do HTML5 e layout centralizado.

---

## 🛠️ Tecnologias Utilizadas

- ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) **HTML5** — estrutura e elemento de áudio
- ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white) **CSS3** — estilização visual (cor de fundo, borda de imagem)

---

musica.html/

---

## 🚀 Como Visualizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/MIcodebot/musica.html.git
   ```
2. Abra o arquivo `index.html` no seu navegador.
3. Clique em **Play** no player de áudio para ouvir a faixa Lofi.

---

## 📚 Conceitos Praticados

- Estilização CSS com `<style>` na mesma página
- Propriedades CSS: `text-align`, `background-color`, `color`, `border`
- Tags semânticas: `<header>`, `<section>`
- Elemento de áudio HTML5: `<audio src="..." controls>`
- Inserção de imagem com borda estilizada
